Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM. such as

  • document.getElementsByTagName(TagName) [to get same elements with same Tag name in the DOM]
  • document.getElementsByClassName(className) [to select those elements which r with the same class name
  • document.getElementById(id) [to get an spesific element in the DOM]
  • document.querySelectorAll(className with selector) [to get element with the same class name.]
  • document.querySelector(id or className) [can get an specific element in the DOM]/li>

Question-2: What is the difference between innerHTML, innerText, and textContent ?

difference between innerHTML, innerText and textContent are as follows

  • innerHTML is all the html tags that are inside element including text but innerText is just the text that are inside the element.
  • innerText is just the text which are visible inside the the element but the textContent returns all text including invisible text .. textContent is more secure and powerful than innerText

Question-3: What is event delegation in the DOM?

To do an event on some same elements , apply event on their parent instead of applying on elements separately.

Question-4: What is event bubbling in the DOM?

event bubbling is a mechanism of DOM. it propagates upwards to root through the DOM tree node

Question-5: How do you create, add, and remove elements using JavaScript?

  • 1. to create elements using JavaScript is : document.createElement("tagName")
  • 2.to add elements using JavaScript : using .appendChild('childName') to the parent .
  • 3. to remove elements using JavaScript: use .removeChild(childName) to the parent.
